## 📘 Suffixes in English

### 1. Inflectional Suffixes (۸ عدد – برای ساخت زمان، جمع، مالکیت، مقایسه)

| Suffix | Meaning | Example |
|--------|---------|---------|
| **-s** | جمع | cat → cats |
| **-'s** | مالکیت | Ali → Ali's book |
| **-s** | سوم شخص مفرد (فعل) | run → runs |
| **-ed** | گذشته (با قاعده) | walk → walked |
| **-en** | اسم مفعول (بعضی افعال) | eat → eaten |
| **-ing** | استمراری / اسم مصدر | go → going |
| **-er** | صفت مقایسه‌ای | fast → faster |
| **-est** | صفت عالی | fast → fastest |

---

### 2. Derivational Suffixes (ساخت کلمه جدید از ریشه)

#### ▶️ Noun Suffixes (اسم‌ساز)

| Suffix | Meaning | Example |
|--------|---------|---------|
| **-er / -or** | شخص انجام‌دهنده | teach**er**, act**or** |
| **-tion / -sion** | عمل یا نتیجه | atten**tion**, televi**sion** |
| **-ness** | حالت یا کیفیت | happi**ness** |
| **-ity / -ty** | حالت یا کیفیت | creativ**ity** |
| **-ment** | نتیجه یا فرآیند | enjoy**ment** |
| **-ance / -ence** | حالت یا فرآیند | import**ance**, differ**ence** |
| **-age** | نتیجه | pack**age** |
| **-al** | عمل | arriv**al** |
| **-ing** | عمل یا نتیجه | paint**ing** |
| **-ism** | اعتقاد / آیین | capital**ism** |
| **-ist** | شخص | art**ist** |
| **-ship** | وضعیت / گروه | friend**ship** |
| **-ure** | فرآیند یا نتیجه | depart**ure** |

#### ▶️ Adjective Suffixes (صفت‌ساز)

| Suffix | Meaning | Example |
|--------|---------|---------|
| **-able / -ible** | قابل… | break**able**, vis**ible** |
| **-al** | مربوط به | music**al** |
| **-ful** | پر از | hope**ful** |
| **-less** | بدون | hope**less** |
| **-ous / -ious** | دارای کیفیت | danger**ous**, cur**ious** |
| **-ive** | دارای ماهیت | act**ive** |
| **-ic** | شبیه به | hero**ic** |
| **-ish** | تا حدودی | child**ish** |
| **-y** | پر از / تا حدودی | cloud**y** |

#### ▶️ Verb Suffixes (فعل‌ساز)

| Suffix | Meaning | Example |
|--------|---------|---------|
| **-ize / -ise** | ایجاد کردن | organ**ize** |
| **-ify** | تبدیل شدن | simpl**ify** |
| **-en** | باعث شدن | lengthen |
| **-ate** | باعث شدن | activ**ate** |

#### ▶️ Adverb Suffix (قیدساز)

| Suffix | Meaning | Example |
|--------|---------|---------|
| **-ly** | به شکلی | quick**ly** |

---

### نکات سریع (Quick Tips)

- **Inflectional suffixes** → ۸ عدد، بدون تغییر دستهٔ کلمه (مثلاً صفت می‌ماند صفت).
- **Derivational suffixes** → می‌توانند دستهٔ کلمه را عوض کنند (مثل slow (صفت) → slowness (اسم)).
- برای صفت‌های مقایسه‌ای و عالی: یک‌هجایی → -er / -est, دو هجایی با -y → -ier / -iest.
