# learn-english

My personal repo for learning English.

## Structure

```
learn-english/
│
├── grammar/                              # English grammar notes
│   ├── README.md                         # List of grammar topics
│   ├── tenses.md                         # Tenses
│   ├── conditionals.md                   # Conditionals
│   └── ...
│
├── vocabulary/                           # Vocabulary from books, documents, etc.
│   ├── general.json                      # General words (Source JSON (manual edit))
│   ├── general.md                        # General words (Generated Markdown (do not edit))
│   ├── deep-learning-with-pytorch.json   # Words from Deep Learning with PyTorch
│   ├── deep-learning-with-pytorch.md     # Words from Deep Learning with PyTorch
│   └── ...
│
└── tools/                                # Helper scripts
    ├── highlights-extractor.js           # Extract highlights from Firefox PDF viewer
    ├── json_to_md.py                     # Convert JSON vocabulary to Markdown
    └── ...
```
